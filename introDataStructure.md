# DataStructure

A data structure is a collection of values.

Datastructure + Algorithms = Programs

## Data Types

### Primitive

They define immutable values – which can’t be changed.

- number
- boolean
- string
- undefined
- null
- symbol

### Non-Primitive

- Object
